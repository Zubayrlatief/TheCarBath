# Elite ParkWash

Premium on-site car care for business parks and premium automotive clients.

## Run locally

```bash
npm install
npm run dev
```

Then open the server URL shown in the terminal (default: http://localhost:5173).

## Tech
- Vue 3 + Vite
- Vue Router
- Black-and-white theme (see `src/assets/main.css`)

## Pages
- Home (`/`)
- Services (`/services`)
- Corporate (`/corporate`)
- Contact (`/contact`)
